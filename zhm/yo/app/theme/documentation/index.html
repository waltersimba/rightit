Home Slider

Default home slider

For full width slider, add navbar-full-slider class to navbar. Then use container-fluid for full slider

Product Listing

Product list use inside col-x class. For example below use all grid using col-xs-12 class. Then use col-md-2 to make 6 'box product' per row.

Use Bootstrap Grid for more information

Product Detail
